# Yei 출시 후 마케팅·세일즈 플레이북 (초안)

> 출시 직후~초기 성장 단계 실행 가이드. 유료 광고보다 **유기적(ASO·SEO·평점)** 중심.
> 가격·키워드는 시작점이며, 데이터를 보고 조정.

## 1. 가격 설계 (티어)
**v1 (한국, 지금)**
- 무료: 국내 규격 5회 저장 (인화 시트·해외 규격 잠금)
- 프리미엄 1회 ₩2,900: 무제한 저장 + 인화 시트

**v1.1 (해외/미국 규격 추가 후) — 2티어 권장**
- 기본 ₩2,900: 국내 무제한 + 인화 시트
- 프리미엄 ₩6,900(~$5): + 해외·비자 규격 (지불의사 높은 비자 수요 겨냥)
- (단순하게 갈 거면: 단일 프리미엄을 ₩4,900으로 올려 해외 규격 포함)

**전환 레버**: "한 번에 통과 · 불합격 시 환불" 보장(leqi 검증). 비자 티어에 특히 효과. (환불은 스토어 정책/수동으로 운영.)
**지역 가격**: 플레이 가격 템플릿으로 시장별 현지가(예: 미국 비자 $4.99).

## 2. ASO (앱스토어 최적화) — 가장 중요한 무료 채널
- **제목/부제**에 핵심 키워드: 예) "Yei – 여권·증명사진·비자사진"
- **키워드(한국)**: 여권사진, 증명사진, 반명함, 비자 사진, 미국 비자 사진, 여권사진 규격, 면허증 사진, 증명사진 만들기
- **키워드(미국, 확장 시)**: passport photo, passport photo maker, visa photo, 2x2 photo, ID photo, DV lottery photo
- **키워드(일본)**: 証明写真, パスポート写真, ビザ写真, 履歴書写真
- **스크린샷**: 앞 2장이 승부. 가치 강조 — 규격 정확 / 흰 배경 / 인화 시트 / 합격 안심, 텍스트 오버레이.
- **현지화**: 진출 시장별 리스팅 번역.

## 3. SEO 콘텐츠 (검색 유입 깔때기)
- 고의도 검색을 잡는 가이드 글 → 앱으로 유입. 예: "미국 비자 사진 규격·만드는 법", "여권사진 규격 총정리", 나라별.
- 간단한 사이트/블로그에 호스팅(FAQ·개인정보처리방침도 같은 곳에).
- 검색해서 들어온 사람은 의도가 높아 전환율이 좋음.

## 4. 평점·리뷰 엔진
- 앱 내 평점 요청을 **만족 순간(저장/인화 성공 직후)**에, 과하지 않게(네이티브 in-app review).
- 리뷰에 답글(초기 수동) → 평점↑ → 랭킹↑ → 설치↑.
- 불만/버그는 빠르게 업데이트로 대응(업데이트 무료).

## 5. 타이밍·시즌
- 수요 급증기에 맞춰 노출·피처링: 유학(F-1) 시즌, 여행 성수기, 여권 갱신 주기.

## 6. 채널 우선순위
1. ASO (무료, 최우선)
2. SEO 가이드 콘텐츠
3. 리뷰·평점
4. 커뮤니티(유학·비자·여행 카페 등) — 도움되는 참여, 스팸 금지
5. (나중) B2B — 유학원·비자 대행사·이민 변호사 묶음/리퍼럴

> 유료 광고는 후순위: 싼 1회 결제라 보통 본전(LTV<CAC)이 안 나옴. 검증된 LTV가 생기기 전엔 지양.

## 7. 측정 (Play Console + RevenueCat)
- 설치, 스토어 전환(방문→설치), **설치→유료 전환**, ARPU, 평점 — 키워드·시장·국내/해외 규격별로.
- 전환율이 낮으면 게이트/가격/스크린샷 A/B.

## 8. 자동화는 나중에 (에이전트 후보)
초기엔 수동으로 이 플레이북을 돌리며 뭐가 먹히는지 학습. 볼륨·다시장이 되면 아래를 에이전트로:
- 리뷰 모니터링 + 다국어 답글 초안(사람 승인)
- 주간 지표 다이제스트(Play + RevenueCat → 요약·이상 감지)
- 시장별 리스팅·SEO 글 초안 대량 생성
- 문의 이메일 분류·초안

> 각각 볼륨이 생겨야 본전. 그 전엔 이 가이드로 충분.
