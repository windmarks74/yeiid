## 4. 차후 (해외 확장 + 미뤄둔 기술 과제)

### 4-A. 확장 로드맵 — 규격 커버리지 (티어)

> **핵심 원칙:** "몇 개국이냐"는 목표가 아님. 여권 사진은 ICAO Doc 9303 베이스라 **크기는 3개 템플릿**(35×45mm / 미국 2×2in / 캐나다 50×70mm)으로 대부분 묶임 → 150개국도 150배 일이 아님.
> **진짜 일 = 해자 = 나라별 파라미터 매트릭스를 정확·최신으로 유지하는 것:** 머리 비율 · 배경색/톤 · 안경 규칙 · (여권≠비자≠ID) · **디지털 업로드본**(픽셀 범위+파일 크기 밴드+포맷). 경쟁 앱이 거부당하는 지점이 바로 이 디테일.
> **금지선:** "보장/100% 통과/공인" 워딩 금지 → "규격에 맞게, 한 번에 통과하도록"까지만. 틀린 규격 = 기관 거부 = 브랜드 최악 실패.
> **세일즈 포인트(정직):** 1회 결제 유지 + "평생 무료 규격 업데이트(규정 바뀌면 앱 업데이트로 자동 반영)" = 구독 없이 결제 욕구 ↑.
>
> ▸ **인프라 현황(2026-06):** 나라 프리셋 테이블 + 제한모드(`restricted`) + 프리미엄 잠금(`premium`) + 눈높이 밴드 + **디지털 용량캡(`maxKB`)** + 인화시트 = **전부 구현됨**. 신규 규격 = `src/usage.ts`에 데이터 1항목 + 공식 출처 검증.
> ▸ **규격 데이터 매트릭스: `SPEC_MATRIX.md`** (10개국 인쇄·디지털·편집·안경·배경·출처 + 구현 매핑 메모). 개발 착수 시 근거 데이터.
> ▸ **정기 점검: `MAINTENANCE.md`** (규격·신청URL 6개월 점검 체크리스트 — 규격/URL은 자주 바뀜 = 운영 핵심. 예: 2026 공무원 gosi.kr→gongmuwon.gosi.kr 이전).

- [ ] **Tier 1 — 35×45mm 흰배경 묶음** (템플릿 1개 + 파라미터로 30~50개국): 셰겐/EU · 영국 비자 · 일본 · 호주 · 아시아 다수. ▸ **한국·셰겐 보유**
- [ ] **Tier 2 — 구분되지만 표준 · 고수요**: 미국 2×2(▸**보유**) · 캐나다 50×70 · 인도(3종) · 중국 33×48(아래)
- [ ] **Tier 3 — 하드 테일**: 배경색 특수(독일 회색 · 프랑스 · 영국 여권 회색/크림) · 디지털 전용 포맷(베트남 eVisa JP2) · 옆얼굴(아르헨티나 3/4) · 엄격 온라인 체커
- [ ] **추천 1차 세트(~10개국)** — 한국 + 미국 · 셰겐 · 영국 · 일본 · 중국 · 인도 · 캐나다 · 호주 ▸ 현재 3/9 (한국·미국·셰겐)
- [ ] 각 규격마다 **공식 출처 URL 앵커** + 최종확인일 기록 (변경 잦음 — 유지보수가 운영 핵심)

### 4-B. 기술 · 컴플라이언스 과제

- [x] 모바일 배경 제거 정공법 — 네이티브 분할(ML Kit Selfie Segmentation) 플러그인
- [x] 나라별 프리셋 시스템(데이터 기반): 미국 2×2 · 셰겐 추가
- [x] 나라별 규정 테이블(배경색·머리 비율·편집 규칙) — 여권 경고 로직 일반화(`isRegulated`)
- [ ] **중국 규격팩** (강력 후보 — 까다로운 규격이라 경쟁 앱 실패 多). ▸ 인프라 준비됨, 데이터+검증이면 됨
  - 인쇄 **33×48mm**, 머리 28~33mm · 너비 15~22mm · 상단 여백 3~5mm, 순백 배경
  - **디지털 업로드본 별도**: 354×472 ~ 420×560px, JPEG **40~120KB**(여권 30~80KB), 눈 간격 ≥60px, 머리 각도 ±20°/±25°
  - 난이도 = (a) 파일 크기/압축(경쟁 앱 거부 지점 — `maxKB`로 대응 가능) (b) 인쇄+디지털 2종 출력
  - "합성 불가" → 중국 모드 AI 배경교체·뽀샤시 **OFF** (KR 여권·미국·셰겐 제한모드 패턴 재사용)
  - 출처 앵커: 주한 중국대사관(kr.china-embassy.gov.cn) — 변경 잦음(규격 DB = 해자)
- [ ] i18n(영어 먼저) + 언어별 스토어 리스팅 ▸ "미국/글로벌 출시 시 재논의"로 보류
- [ ] 출시 국가 설정 + 지역별 가격
- [ ] 지역 기반 자동 UI(다운로드 지역→해당국 기본 규격) ▸ 글로벌 출시 시 (기기 locale 방식)
- [ ] EU DSA 트레이더 등록(이름·주소·연락처 공개 — 주소 노출 주의)
- [ ] GDPR · CCPA 대응
- [ ] (선택) iOS — App Store(연 $99), Capacitor iOS, 심사

### 4-C. UX·카피 아이디어 (v12 · 전환·가치) — 코드 변경이라 출시 후 본인 확인 후

- [ ] **면허/일반 모드 "젊어 보이게" 팁**: "잡티 살짝 완화 + 깔끔한 배경으로 인상이 산뜻·젊어 보여요" (`notice`/마이크로카피). **여권은 제외**(restricted=원본 보존). 가드레일: "자연스럽게, 본인 확인되는 선". ▸ 포지셔닝 근거: `docs/yei-marketing-playbook.md` §1-B.
- [ ] **무료횟수 마이크로카피**: "편집은 무제한 무료 — 저장할 때만 1장 차감" → panic-save↓, 체감 인색함 완화. (카운트는 `recordDownload`로 갤러리 저장 성공 시에만 +1 — 편집·미리보기는 무료.)
- [ ] **에디터 전/후 비교**(또는 미리보기 신뢰도↑) → 비교하려고 갤러리에 여러 장 저장(=카운트 소모)하는 행동을 줄임.
- [ ] **무료횟수 정책**: 출시 **5회 유지**(신규앱 평점 보호). "페이월 노출률"·"첫 페이월까지 저장수" 계측 후 조정 — 감 아닌 데이터. (3회 인하는 학습 중 페이월 → 악평 위험 → 비권장.)
