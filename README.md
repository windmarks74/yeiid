# 증명사진 만들기 — 여권 · 운전면허증

브라우저에서 동작하는 한국 증명사진 편집 웹앱. 여권/운전면허증 사진을 규격에 맞게
크롭하고, 가벼운 보정·효과를 적용하고, 파일 용량을 맞춰 내려받습니다.

**개인정보 보호:** 사진은 기기 밖으로 전송되지 않고 **전부 브라우저(클라이언트)에서 로컬 처리**됩니다.
(배경 제거 AI 모델 파일만 최초 1회 다운로드 — 사진 자체는 절대 전송되지 않음)

## 실행 방법

```bash
npm install
npm run dev
```

→ 브라우저에서 http://localhost:5173/ 접속

빌드:

```bash
npm run build      # 타입체크 + 프로덕션 빌드 (dist/)
npm run preview    # 빌드 결과 미리보기
```

## 기능

- **사진 가져오기** — 파일 업로드 (모바일은 카메라 촬영도 선택 가능)
- **용도 선택** — 여권 / 운전면허증 / 일반
  - **여권 모드**: 무보정 원칙 규정 배너 표시. 배경 교체·뽀샤시를 켜면 *반려 가능* 경고
    (외교부 기준상 과도한 보정·배경 합성은 반려 사유). 권장 범위는 크롭·규격·용량 +
    피부톤을 왜곡하지 않는 가벼운 밝기/노출/화이트밸런스 보정까지.
  - **면허증 / 일반**: 배경 교체·효과 자유
- **규격 프리셋** — 한국 여권·면허증 35×45mm (413×531px @300DPI), 미국 여권 51×51mm (600×600px)
- **크롭 & 정렬** — 비율 고정 크롭(드래그 이동 + 확대 슬라이더), 얼굴 가이드(정수리선 +
  머리 70~80% 턱 밴드 + 중앙선) 오버레이
- **가벼운 보정** — 노출 · 밝기 · 화이트밸런스 (여권 허용 범위, 미리보기=결과 일치)
- **효과** — 뽀샤시/글로우(블러+소프트라이트 합성, AI 아님), 배경 제거 후 단색 배경 교체
  (흰색/연하늘/회색), 배경 모델 품질 선택(빠름 fp16 / 고품질 isnet)
- **용량 맞추기** — "목표 KB 이하" 지정 시 JPEG 품질을 이분 탐색해 자동 조절, 결과 용량 실시간 표시
- **내려받기** — 규격 정보가 포함된 파일명(JPEG)

## 기술 스택

- **React + TypeScript + Vite**
- **Canvas API** — 크롭 / 보정 / 효과 / 리사이즈 / JPEG 압축 (미리보기와 출력이 동일 파이프라인)
- **@imgly/background-removal** — 배경 제거(로컬 WASM, onnxruntime)

## 배경 제거 모델 (완전 오프라인)

- 배경 제거 모델(빠름 fp16 + 고품질 isnet)과 onnxruntime WASM은 `public/imgly/` 에 **로컬 번들**되어
  같은 출처에서 서빙됩니다. **인터넷 없이 동작**하며, 사진은 물론 모델도 외부로 나가지 않습니다.
- 모델 파일은 약 300MB라 저장소에 포함하지 않습니다. **최초 1회** 아래로 받으세요:
  ```bash
  node scripts/fetch-model.mjs   # public/imgly/ 에 모델 다운로드 (한 번만)
  ```
- 멀티스레딩(빠른 추론)을 위해 개발/프리뷰 서버에 **교차 출처 격리 헤더(COOP/COEP)**가 설정돼 있습니다
  (`vite.config.ts`). 별도 호스팅에 배포할 경우 그 서버에도 같은 헤더가 필요합니다.
- `npm run build` 시 `public/imgly/` 가 `dist/` 로 복사되어 배포본도 오프라인 동작합니다.

## 구조

```
src/
  App.tsx        용도/규격/보정/효과/출력 상태 + UI 패널
  Cropper.tsx    팬·줌 크롭 + 얼굴 가이드 (캔버스 미리보기)
  presets.ts     규격 정의 + mm↔px 환산
  usage.ts       용도(여권/면허증/일반) + 규정 헬퍼
  adjust.ts      가벼운 보정 (노출/밝기/화이트밸런스) 픽셀 연산
  effects.ts     배경 색 + 뽀샤시/글로우 캔버스 합성
  bg.ts          배경 제거 (모델 동적 로드)
  imageUtils.ts  이미지 로드 / 렌더 파이프라인 / 목표 용량 인코딩
```
